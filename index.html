My Work

Avocardio

Avocardio is a fitness project built using the React framework. Within the project, users have the ability to search for exercises based on various criteria such as body part, muscle group, and equipment. The search results are presented in GIF format, allowing users to easily follow along with the exercise demonstrations. To provide the exercise data and power the search results, the project utilizes the ExerciseDB API.

Weather Dashboard

This app, Weather dashboard, allows the users to search for a city and view its current and future weather conditions, including temperature, humidity, and wind speed. The dashboard stores search history and allows users to easily access the weather information for previously searched cities.

Calendar

A planner app featuring current day display, time blocks for business hours, color-coded blocks for past/present/future, event entry on click, local storage save, and persistent event data after page refresh.
